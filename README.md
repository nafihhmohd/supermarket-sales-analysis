# Sales Analysis Project

## Description
Data analysis project exploring retail sales to uncover insights on branch performance, product popularity, customer behavior, and payment trends. The project uses Python and data visualization in Google Colab to identify key trends, top-performing products, and actionable recommendations to boost revenue and customer satisfaction.

## Dataset
The project uses a dataset containing retail transaction details, including:
- Branch information
- Product categories
- Payment methods
- Customer ratings
- Revenue and transaction counts

## Objectives
- Determine the most profitable branches and product categories.
- Analyze customer preferences and ratings.
- Explore payment method trends and their impact on sales.
- Identify top-performing products and potential areas for growth.
- Provide actionable insights and recommendations.

## Tools & Technologies
- Python (Pandas, Matplotlib, Seaborn)
- Google Colab
- Git & GitHub

## Key Insights
- Branch C generates the most revenue.
- E-Wallet is the most used payment method.
- Health & Beauty is a strong product line.
- Customer ratings are generally high.

## Next Steps
- Analyze sales trends over time (monthly/weekly).
- Segment customers based on purchase behavior.
- Compare branch performance using multiple metrics (average transaction value, revenue, ratings).
- Explore correlations between payment methods and product categories.
- Create dashboards for visual representation of insights.
